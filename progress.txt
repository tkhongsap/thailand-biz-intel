- Build command: npx next build
- Dev: next dev -p 3300

## Learnings
- Amphur names in DB have "อ." prefix on some rows — normalizeAmphurName() strips it
- Function declarations inside try blocks fail TS strict mode — use arrow functions
- Map iteration with `for...of` requires downlevelIteration or ES2015 target — use Array.from().forEach()
- GeoJSON features have centroid in properties — access via `(props as Record<string, unknown>).centroid`
- Canvas charts need DPR scaling for retina displays
- API route: `export const dynamic = "force-dynamic"` for SQLite-backed routes
